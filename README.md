# Analisis-Banjir-Berbasis-GEE
Ini adalah skrip Google Earth Engine (GEE) untuk deteksi banjir menggunakan fusi data SAR (Sentinel-1) dan Optik (Sentinel-2). Logika yang Anda gunakan—membandingkan backscatter sebelum (baseline) dan sesudah (event) kejadian—adalah metode standar industri untuk pemetaan banjir cepat.
